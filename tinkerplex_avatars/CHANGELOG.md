# Changelog

## 1.0.0

Initial release of TinkerPlex Avatars package.

### Features

- `UserAvatar` widget for displaying DiceBear avatars
- `AvatarPicker` widget for selecting avatar styles
- `ProfileEditor` widget for full profile editing
- `TinkerPlexProfileService` for Supabase profile CRUD
- `UserProfile` model
- 7 CC0-licensed avatar styles:
  - pixel-art
  - lorelei
  - thumbs
  - avataaars
  - bottts
  - initials
  - shapes
